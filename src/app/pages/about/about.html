About

Simple learning portal for students.

The Online Learning Portal is created as an Angular routing assignment. It shows how different pages can be opened without reloading the whole application.

Concepts Covered

  • Angular Router configuration
  • Navigation menu
  • Router links
  • Default and fallback routes

How It Works

Routes

Each URL path is connected with a page component in the route file.

Navigation

The menu uses router links so users can move between pages smoothly.

Router Outlet

The selected page is displayed inside the main layout using router outlet.
